Lumi — Privacy Policy

How Lumi handles data and protects your privacy.

Last updated: 21 November 2025

This Privacy Policy explains what data Lumi (“the Bot”, “we”, “us”) collects and how it is used.

By using Lumi, you agree to the collection and use of information as described below.

1. Data We Collect

Lumi only collects data that is necessary to provide its features. This may include:

1.1 Discord Identifiers

  • User IDs and usernames
  • Server (guild) IDs
  • Channel IDs
  • Message IDs

These identifiers are required to know who used the Bot and where.

1.2 Message Content & Settings

Depending on which features you use, we may temporarily or persistently store:

  • Command usage (for logs or cooldowns)
  • Custom responses / auto-replies
  • Birthdays, warnings, moderation notes, or similar server-specific data
  • Configuration such as:
    • Prefix
    • Enabled/disabled features
    • Role IDs, channel IDs used by Lumi

We do not intentionally store private or sensitive information beyond what is necessary for the Bot’s functionality.

Please avoid using Lumi commands to store passwords or sensitive personal data.

1.3 Technical Data

Our hosting provider may log technical information such as:

  • IP address of the server where the bot is hosted
  • Error logs and performance data

This is used solely to keep Lumi online and debug issues.

2. How We Use Your Data

We use the collected data to:

  • Execute commands and provide features you request.
  • Remember server settings and preferences.
  • Prevent abuse (for example, rate limiting or blacklisting).
  • Monitor errors and improve stability.

We do not sell your data or share it with advertisers.

3. Data Sharing

We may share limited data:

  • With Discord, as part of normal Bot operation (e.g. sending messages, embeds, etc.).
  • With our hosting provider, to run Lumi.
  • When required by law, if we receive a valid legal request.

We do not share your data with random third parties.

4. Data Retention

  • Configuration and server-related data are stored as long as Lumi is in your server or until they are manually reset.
  • Some logs may be kept for a limited time for debugging and abuse prevention.
  • If Lumi is removed from a server, we may delete associated configuration data over time during clean-ups.

You can request manual deletion of specific stored data (for example, birthdays or auto-replies) by contacting us.

5. Data Access & Deletion

If you want to:

  • Know what data Lumi has stored about your server or user, or
  • Request deletion of certain data

You can contact us anytime using the information provided in the Contact section below.

6. Children’s Privacy

Lumi is intended for use on Discord and follows Discord’s minimum age requirements.

We do not knowingly collect personal information from children under that minimum age.

If you believe that data about a minor has been stored, please contact us so we can delete it.

7. Security

We take reasonable technical steps to keep stored data secure (such as restricted access and basic server protections), but:

  • No system is 100% secure.
  • We cannot guarantee absolute security of any stored information.

8. Changes to This Policy

We may update this Privacy Policy from time to time.

  • Significant changes may be announced in the support server or via a status message.
  • Continued use of Lumi after changes take effect means you accept the new Policy.

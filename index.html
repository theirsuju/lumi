AI system

All AI-powered features and helpers.

Configure Lumi's AI behaviour with these commands:

.ai enable [#channel]
Enable AI in a channel. If no channel is given, it uses the channel where you run the command.
.ai disable [#channel]
Disable AI in a specific channel (good for spammy / games channels).
.ai enable server
Turn AI on for the whole server. All channels can use AI except ones you‚Äôve disabled.
.ai disable server
Turn AI off for the whole server. Only channels you explicitly enabled will still be able to use it.
.ai status
Show current AI settings for the server and for the channel you are in.

Moderation commands

For staff and mods to manage the server.

Core moderation tools to keep your server safe and clean:

Prefix control
.setprefix <prefix>
Changes Lumi‚Äôs prefix only for this server. Example: .setprefix ! ‚Üí commands become !help, !ai, etc. (Admins only)
.prefixreset
Resets the prefix back to the default . for this server. (Admins only)
Kick
.kick @user [reason]
Kicks the mentioned user from the server with an optional reason.
Mute / Timeout
.mute @user <duration> [reason]
.timeout @user <duration> [reason]
Temporarily mutes / timeouts the user for the given duration (e.g. 10m, 1h, 2d).
.unmute @user [reason]
Removes any active mute / timeout on the user.
Nickname tools
.nick @user <nickname>
Changes the nickname of the mentioned user.
.nickreset @user
Resets the user's nickname back to their original username.
Slowmode
.slowmode <seconds> [channel]
Enables slowmode on the channel (or mentioned channel) for the chosen seconds.
.slowmode off
Turns off slowmode for the current channel.
Emoji stealer
.steal [emoji]
Adds the provided custom emoji to the server with the same name. (Works with animated emojis too)
Purge / clear
.c <amount>
Deletes the last amount messages in the channel (default is 1 if no number is given).
.c @user <amount>
Deletes recent messages from a specific user only.
.c bots <amount>
Deletes messages sent by bots.
.c humans <amount>
Deletes messages sent by human users only.
.c links <amount>
Clears messages containing links (good for quick anti-raid).
.c invites <amount>
Deletes messages that contain Discord invites.
.c contains <text> <amount>
Deletes messages that contain the given text phrase.
Cleanup
.cleanup / .cl / .clean
Cleans up Lumi‚Äôs own command replies / clutter from the channel.
Warn system
.warn <user/id> [reason]
Adds a warning to the user with an optional reason.
.warnings <user/id>
Shows all stored warnings for that user.
.warnings clear <warnID>
Clears a single warning using its ID.
.warnings clearall <user/id>
Removes every warning stored for that user.
Role tools
.role <user> <role>
Gives or removes the specified role from the user (mention / ID / name supported).
.rolecreate <name> <#hex>
Creates a new role with the given name and colour.
.rolerename <role> <newname>
Renames an existing role.
.roledelete <role>
Deletes the specified role (be careful!).
Nuke
.nuke [channel]
Clones and resets a channel, wiping all messages. If no channel is given, nukes the current channel.
Lockdown
.lock / .unlock
Locks or unlocks the current channel for regular members.
.lockall / .unlockall
Quickly locks or unlocks all main channels during raids.
Ban tools
.ban @user [reason]
Bans the mentioned user from the server, optionally with a reason.
.unban @user
Unbans the user (works with username+discriminator or ID, depending on setup).
.forceban @user [reason] / .hackban @user [reason]
Bans a user by ID even if they are not in the server yet.

All moderation commands should be used by trusted staff only. Make sure roles and permissions are configured correctly.

Utility commands

Useful tools, info and shortcuts.

Handy tools for help, pings, giveaways, info and auto systems:

Basic

Shows Lumi‚Äôs help & links embed inside Discord.

Shows all birthday commands and formats.
